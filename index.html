@beyond/submission

This is a micro-frontend module that needs to be loaded into a Single-SPA application.

Integration Steps

1. Add the module to your import map:

{
  "imports": {
    "@beyond/submission": "http://localhost:3000/beyond-submission.js"
  }
}

2. Register the application in your Single-SPA root config:

registerApplication({
  name: "@beyond/submission",
  app: () => System.import("@beyond/submission"),
  activeWhen: ["/submission"]
});

Development

The module is being served at: http://localhost:3000/beyond-submission.js
